System Layers

Browser Vue 3 SPA — port 3000 — 7 views, FilterBar, modals
State useFilters() composable — singleton shared across all views
HTTP api.js — Axios, builds URLSearchParams, skips 'all' values
API FastAPI — port 8001 — CORS open, Pydantic response models
Logic main.py — apply_filters(), filter_by_month()
Data mock_data.py — loads 6 JSON files into memory at startup

Data Files — server/data/

inventory.json
Stock items — id, sku, name, category, warehouse, qty_on_hand, reorder_point, unit_cost, location
orders.json
Customer orders — order_number, customer, items[], status, warehouse, category, order_date, expected_delivery, total_value
demand_forecasts.json
Demand forecasts — item_sku, current_demand, forecasted_demand, trend (increasing/stable/decreasing), period
backlog_items.json
Shortage items — order_id, item_sku, qty_needed, qty_available, days_delayed, priority (high/medium/low)
spending.json
Financial data — spending_summary (costs + % changes), monthly_spending[], category_spending[]
transactions.json
Recent transactions — displayed in Spending view
purchase_orders.json
POs for backlog items — supplier_name, qty, unit_cost, expected_delivery_date, status (currently empty)
